# Devcamper API

> Backend API for Devcamper application which is a bootcamp directory website created only for learning purposes

## Usage

Rename "config/config.env.env" to "config/config.env" and update values/settings to your own

## Install Depedencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
